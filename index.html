Welcome to Employee Management System Project

This project is a Python desktop application that runs locally on your computer.

About the Project

I’ve built an Employee Management System (EMS) using:
>Python with tkinter and customtkinter for the GUI
>MySQL for data storage (pymysql for database interaction)
>Modular structure with three main files:
1.) login.py – Login interface with simple credential validation
2.)ems.py – Full EMS dashboard for CRUD operations
3.)database.py – Backend database operations (insert, fetch, search, update, delete)

How to Run the Project

To use this app on your computer, follow these steps:

  1. Make sure you have Python installed. You can download it from python.org.
  2. Download or clone this repository to your computer.
  3. Open a terminal or command prompt and navigate to the project folder.
  4. Run the main Python file. For example:
python ems.py
